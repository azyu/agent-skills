# agent-skills

Local agent skills repository.

## Structure

- `.agents/skills/instruction-writer`: skill for drafting and rewriting instruction documents into clearer, model-friendly rules
- `.agents/skills/gemini-google-web-search`: skill for current web research through Gemini CLI's `google_web_search` tool
- `.agents/skills/shell-script`: skill for creating, editing, and reviewing shell scripts with shell-specific guidance for bash, sh, and zsh
